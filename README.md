# PCVerse

An interactive 3D educational guide about PC components, built with React, Vite, Three.js, React Three Fiber, and Tailwind CSS.

## Features
- **3D Interactive Scene**: A stylized 3D model of a PC with clickable components.
- **Exploded View**: An animated state separating all parts for better visibility.
- **Camera Controls**: Orbit around the PC, or click a component to automatically zoom into it.
- **Educational Info Panel**: Detailed information about each part, its role, fun facts, and its impact on system performance.
- **Responsive Design**: Adapts for both desktop and mobile users.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL provided by Vite (usually http://localhost:5173).
